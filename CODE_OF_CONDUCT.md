# Code of Conduct

We want AgentCrew to be a welcoming project for contributors of all
backgrounds and experience levels.

- Be respectful and assume good intent.
- No harassment, discrimination, or personal attacks.
- Disagreements about code or design are fine and expected — keep them
  focused on the work, not the person.
- Maintainers may remove comments, commits, or contributors that violate
  this code, and may report unacceptable behavior as appropriate.

Questions or reports can be raised via a GitHub issue or directly to the
project admin.
